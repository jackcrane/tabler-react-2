## Margin directions

The margin utilities can be applied to the following directions:

- `mt-*` = `margin-top`
- `mb-*` = `margin-bottom`
- `m-*` = `margin`

## Margin sizes

The margin utilities can be applied to the following sizes:

- `0` = `0`
- `1` = `.25rem`
- `2` = `.5rem`
- `3` = `1rem`
- `4` = `2rem`
- `5` = `4rem`
- `auto` = `auto`
